# Popupの表示方法

## Popupを表示する方法

### 方法1: 拡張機能のアイコンをクリック（推奨）

1. Chrome拡張機能を読み込む
2. ブラウザのツールバーに拡張機能のアイコンが表示されます
3. アイコンをクリックするとpopupが表示されます

**注意**: 現在アイコンファイルが設定されていないため、デフォルトのアイコンが表示されます。

### 方法2: 拡張機能の管理ページから

1. `chrome://extensions/` を開く
2. 「U-FRET Ad Blocker」の拡張機能を見つける
3. 拡張機能の右側にある「アクション」ボタン（三点リーダー）をクリック
4. 「ポップアップを開く」を選択

### 方法3: 直接URLで開く（開発用）

開発中は、以下のURLでpopupを直接開くことができます：

```
chrome-extension://[拡張機能ID]/popup.html
```

拡張機能IDは `chrome://extensions/` で確認できます。

## Popupの機能

現在のpopupには以下の情報が表示されます：

- **ステータス**: 広告ブロックが有効であることを表示
- **削除された要素数**: Content Scriptから取得した削除された要素の数
- **説明**: 拡張機能の動作についての説明

## トラブルシューティング

### Popupが表示されない場合

1. `manifest.json` の `action.default_popup` が正しく設定されているか確認
2. `popup.html` ファイルが存在するか確認
3. ブラウザのコンソールでエラーを確認（F12 → Console）

### 統計情報が表示されない場合

1. Content Scriptが正しく読み込まれているか確認
2. ページをリロードしてからpopupを開く
3. ブラウザのコンソールでエラーを確認
