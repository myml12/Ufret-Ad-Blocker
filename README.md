# U-FRET Ad Blocker

U-FRETサイト上の広告要素を自動削除するChrome拡張機能。

## 機能

- Google広告（iframe）の自動削除
- Geniee広告の自動削除
- YADS広告の自動削除
- その他の広告要素の自動削除
- DOM変更の監視による動的広告の削除
- 広告関連スクリプトのブロック
- スクロール無効化の防止

## インストール

### リリース版からインストール（推奨）

1. [GitHubのリリースページ](https://github.com/myml12/Ufret-Ad-Blocker/releases)から最新のリリースを開く
2. `Ufret-Ad-blocker-vX.X.X.zip` をダウンロード
3. ダウンロードしたzipファイルを解凍する
4. Chromeで `chrome://extensions/` を開く
5. 「デベロッパーモード」を有効化
6. 「パッケージ化されていない拡張機能を読み込む」をクリック
7. 解凍したフォルダを選択

### 開発版からインストール

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択

## 使用方法

インストール後、自動的に動作。U-FRETサイトを開くと各ページで広告要素が自動削除される。

## 削除対象

`ref/eliminate.md`に記載の要素を削除。

- Google広告のiframe要素
- Geniee広告のオーバーレイ要素
- YADS広告要素
- その他の広告要素（IDやクラスパターンで判定）
