# U-FRET Ad Blocker

U-FRETサイト上の広告要素を自動的に削除するChrome拡張機能です。

## 機能

- Google広告（iframe）の自動削除
- Geniee広告の自動削除
- YADS広告の自動削除
- その他の広告要素の自動削除
- DOM変更の監視による動的広告の削除
- 広告関連スクリプトのブロック

## インストール方法

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使用方法

インストール後、自動的に動作します。U-FRETサイトを開くと、広告要素が自動的に削除されます。

## 削除対象

`ref/eliminate.md`に記載されている以下の要素を削除します：

- Google広告のiframe要素
- Geniee広告のオーバーレイ要素
- YADS広告要素
- その他の広告要素（IDやクラスパターンで判定）

## 注意事項

- この拡張機能は教育目的で作成されています
- サイトの利用規約を確認してから使用してください
