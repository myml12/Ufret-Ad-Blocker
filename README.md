# U-FRET Ad Blocker

U-FRETサイト上の広告要素を自動削除するChrome拡張機能。

## 機能

- Google広告（iframe）の自動削除
- Geniee広告の自動削除
- YADS広告の自動削除
- その他の広告要素の自動削除
- DOM変更の監視による動的広告の削除
- 広告関連スクリプトのブロック
- スクロール無効化の防止

## インストール

1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使用方法

インストール後、自動的に動作。U-FRETサイトを開くと広告要素が自動削除される。

## 削除対象

`ref/eliminate.md`に記載の要素を削除。

- Google広告のiframe要素
- Geniee広告のオーバーレイ要素
- YADS広告要素
- その他の広告要素（IDやクラスパターンで判定）

## 注意事項

- 教育目的で作成
- サイトの利用規約を確認してから使用
