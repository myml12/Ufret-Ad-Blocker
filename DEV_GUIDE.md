# U-FRET Ad Blocker 開発ガイド

開発・デバッグ方法の説明。

## 開発環境のセットアップ

### Chrome拡張機能の読み込み

1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトのフォルダを選択

### 拡張機能のリロード

コード変更後は拡張機能をリロード。

1. `chrome://extensions/` を開く
2. U-FRET Ad Blockerの「再読み込み」ボタンをクリック
3. 対象のWebページをリロード（F5キー）

## デバッグ方法

### Content Scriptのデバッグ

Content Script（`content.js`）のデバッグは対象Webページのコンソールで実施。

1. U-FRETサイトを開く
2. 右クリック → 「検証」を選択（またはF12キー）
3. 「Console」タブを開く
4. `content.js`のログやエラーを確認

### Popupのデバッグ

Popup（`popup.html`）のデバッグは拡張機能のポップアップ上で実施。

1. 拡張機能のアイコンをクリックしてポップアップを開く
2. ポップアップ上で右クリック → 「検証」を選択
3. 別ウィンドウでデベロッパーツールが開く
4. `popup.js`のログやエラーを確認

## トラブルシューティング

### 広告が削除されない

1. 拡張機能が有効か確認
2. ページをリロード（F5キー）
3. コンソールでエラーを確認
4. `ref/eliminate.md`に該当要素のセレクタが記載されているか確認

### スクロールが無効化される

1. コンソールで`enableScroll()`を実行
2. ページをリロード
3. 拡張機能を再読み込み

### エラーが発生する

1. コンソールでエラーメッセージを確認
2. `content.js`の該当箇所を確認
3. セレクタが正しいか確認
